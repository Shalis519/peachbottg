# 🌸 Цветок Персика — Telegram-бот по Ба Цзы

Telegram-бот **@Flowerpeach_bot** для расчёта Цветка Персика (桃花) и благоприятных дней активизации по китайской астрологии Ба Цзы.

## Возможности

- Определение Цветка Персика по году и дню рождения
- Расчёт благоприятных дней и часов для активизации на 30 дней вперёд
- Учёт Слияния, Союза и Сезона для всех 4 животных Персика (Крыса, Кролик, Лошадь, Петух)
- Фильтрация дней/часов по Благородному человеку (天乙贵人), пустым и вредным часам
- 5 способов активизации с инструкцией
- Кнопка благодарности (помощь животным и поддержка проекта)

## Стек

- Python 3.11+
- [aiogram](https://docs.aiogram.dev/) 3.13.1 — Telegram Bot API
- [lunar-python](https://pypi.org/project/lunar-python/) — лунный календарь и Ба Цзы
- [pytz](https://pypi.org/project/pytz/) — часовые пояса

## Запуск локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/flowerpeach-bot.git
   cd flowerpeach-bot
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` на основе `.env.example`:
   ```bash
   cp .env.example .env
   ```
   Укажите токен бота в `.env`.

4. Запустите бота:
   ```bash
   cd bot && python bot.py
   ```
